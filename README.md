This code accompanies the paper [Simple SIMON: FPGA implementations of the SIMON 64/128 Block Cipher](FPGA_implementations_of_the_SIMON_64-128_Block_Cipher.pdf) and offers VHDL implementations of various architectural designs of the SIMON 64/128 block cipher as a cryptographic component offering encryption, decryption and self-contained key-scheduling capabilities aimed at the Xilinx Spartan-6 FPGA series.
